# Keywords in Java

In Java, **keywords** are reserved words that have predefined meanings and cannot be used as identifiers (such as variable names, class names, or method names). These keywords are essential to the Java programming language.

---

## List of Java Keywords

### 1. **Data Types**
- `boolean`
- `byte`
- `char`
- `short`
- `int`
- `long`
- `float`
- `double`

### 2. **Control Flow**
- `if`
- `else`
- `switch`
- `case`
- `default`
- `while`
- `do`
- `for`
- `break`
- `continue`
- `return`

### 3. **Access Modifiers**
- `public`
- `private`
- `protected`

### 4. **Class, Object, and Inheritance**
- `class`
- `interface`
- `extends`
- `implements`
- `new`
- `this`
- `super`

### 5. **Exception Handling**
- `try`
- `catch`
- `finally`
- `throw`
- `throws`

### 6. **Modifiers**
- `abstract`
- `final`
- `static`
- `synchronized`
- `volatile`
- `transient`

### 7. **Packages and Imports**
- `package`
- `import`

### 8. **Other Keywords**
- `void`
- `enum`
- `instanceof`
- `assert`
- `native`
- `strictfp`
- `goto` *(reserved but not used)*
- `const` *(reserved but not used)*
